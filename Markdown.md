# Markdown 格式参考指南

## 标题
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 文本样式

| 效果 | 写法 | 示例 |
|------|------|------|
| 粗体 | `**文本**` 或 `__文本__` | **粗体文本** |
| 斜体 | `*文本*` 或 `_文本_` | *斜体文本* |
| 粗斜体 | `***文本***` 或 `___文本___` | ***粗斜体文本*** |
| 删除线 | `~~文本~~` | ~~删除线文本~~ |
| 下标 | `H~2~O` | H~2~O |
| 上标 | `E=mc^2^` | E=mc^2^ |
| 高亮 | `==文本==` | ==高亮文本== |
| 代码 | `` `代码` `` | `code` |

## 列表

### 无序列表
```
- 项目 1
- 项目 2
  - 嵌套项目 2.1
  - 嵌套项目 2.2
- 项目 3
```

### 有序列表
```
1. 第一项
2. 第二项
   1. 嵌套项 2.1
   2. 嵌套项 2.2
3. 第三项
```

### 任务列表
```
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个任务
```

## 链接和图片

### 外部链接
```
[链接文本](https://example.com)
[链接文本](https://example.com "悬停提示")
```

### Obsidian Wiki 链接
```
[[笔记名称]]
[[文件夹/笔记名称]]
[[笔记名称|显示文本]]
[[笔记名称#标题]]
```

### 图片
```
![alt 文本](图片URL)
![alt 文本](图片URL "图片标题")
```

### Obsidian 图片嵌入
```
![[图片文件名.png]]
![[图片文件名.png|宽度]]
```

## 表格

```
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据 | 数据 | 数据 |
| 数据 | 数据 | 数据 |
```

### 对齐方式
```
| 左对齐 | 居中 | 右对齐 |
|:------|:----:|-------:|
| 左 | 中 | 右 |
```

## 脚注

```
这是一个脚注[^1]

[^1]: 脚注内容
```

## 其他常用格式

### 水平线
```
---
```

### 转义字符
```
\*不是斜体\*
\[不是链接\]
```

### HTML 标签（Obsidian 支持）
```
<u>下划线</u>
<mark>高亮</mark>
<small>小字体</small>
```

### 注释（不会显示）
```
<!-- 这是注释 -->
```

## Obsidian 特定语法

### 嵌入笔记
```
![[笔记名称]]
![[笔记名称#标题]]
```

### 嵌入代码块
```
![[笔记名称#^代码块ID]]
```

### 标签
```
#标签名
#标签/子标签
```

### 属性（Frontmatter）
```yaml
---
title: 笔记标题
date: 2024-01-01
tags: [标签1, 标签2]
---
```
